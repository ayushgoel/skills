# Skills

Personal agent skills. Each skill ships as a directory with a `SKILL.md` file (and optional supporting files). Some skills are also packaged as a `.skill` archive for distribution.

## foam-notes

Helps maintain a Foam knowledge graph. **Open the Foam repository as the Cursor workspace** when using this skill so the agent reads and writes notes in the correct project.

Install and enable skills according to Cursor’s current documentation for agent skills.
